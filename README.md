# The Maze Game (with airstrike)
##### Author: Eric Gao
##### Block: C
##### Programming Level 2
#### Jan 21, 2026
## Description

This project is a pygame where control a cat to go around a randomly generated maze trying to find the way to the fish. The player starts from the top left and tries to get to the bottom right where the fish is. 

There is a time limit to the game, and the player loses when they did not make it to the fish under the time limit of 120 seconds.

In addition, there are airstrike explosions that appear every 2 seconds and stays there for 2 seconds. If the player position overlaps with the airstrike explosion, the player loses.

## Controls

- WASD to move up left down right
- Press R to restart the game, only when the game ends (lose or win)
